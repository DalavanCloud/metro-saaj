SAAJ 1.3 Platform Requirements and Dependency on Other Technologies

  The SAAJ 1.3 Reference Implementation currently has the following platform requirements :
  1.  Sun's JDK/JRE 1.4 + Sun's JAXP 1.3 Implementation (Placed in jre/lib/endorsed)
  2.  OR Sun's  JDK/JRE 1.5 or Later
  3.  AND  JavaBeansTM Activation Framework (http://java.sun.com/beans/glasgow/jaf.html)
