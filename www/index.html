SAAJ Standard Implementation Project

Welcome

Welcome to the SAAJ Standard Implementation Project. This is a project in the JWSDP subcommunity of the Java community on Web Services and XML at java.net.

Your participation is fundamental to make this project useful to the Java Community. There are many ways to participate in this project; we encourage you to download and use our implementations.

  • Overview of the SAAJ project
  • Downloads
    • JWSDP: the stable release of the SAAJ SI; JWSDP 1.3 is the latest release.
  • Report a bug
  • Join the users list for general discussion.
  • Read the FAQ!
  • Contribute to the Wiki, collecting uses of the implementation and listing additional platforms.

We will continue to transition more of our development efforts to this site in the future. Some content, including the tutorial, is still only available from the JWSDP home page at java.sun.com.
